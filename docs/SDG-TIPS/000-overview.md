# What Are Tips?

Tips are short one-line reminders that help you discover SDG-OS features and keyboard shortcuts you might forget. They appear automatically when you open a terminal, or you can browse them on demand.

**Ways to see tips:**

| Method | What it does |
|--------|--------------|
| Run `tipme` | Shows a random tip in the terminal |
| Run `sdgtip random` | Same as tipme |
| Run `sdgtip list` | Opens an interactive browser to search through all tips |
